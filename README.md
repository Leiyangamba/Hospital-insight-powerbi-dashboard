# Hospital Insights Dashboard (Power BI + MySQL)

 📌 Project Overview
The Hospital Insights Dashboard is an interactive data analytics project built using Power BI and MySQL.  
This project analyzes hospital operations, patient information, doctor performance, and financial data to provide meaningful insights for decision-making.

The dashboard helps visualize hospital performance through different analytical sections such as patient statistics, doctor workload, hospital operations, and financial analysis.

---

🛠 Tools & Technologies Used
- Power BI
- MySQL
- Excel / CSV Dataset

---

📊 Dashboard Pages

 1. Home Page
The main navigation page that allows users to access different sections of the dashboard.

 2. Overview
Provides a high-level summary of hospital performance including key metrics and trends.

 3. Patient Analysis
Shows insights related to:
- Patient admission trends
- Patient demographics
- Treatment statistics

 4. Hospital Analysis
Analyzes hospital operations including:
- Bed availability
- Department performance
- Hospital capacity

 5. Doctor Analysis
Displays information about:
- Doctor workload
- Department distribution
- Patient treatment statistics

 6. Finance Analysis
Provides financial insights such as:
- Hospital revenue
- Treatment costs
- Financial performance trends
